Unidad 07

🎮 Educar a través del juego: programación (Scratch)

⏱️ Duración estimada: 10-12 sesiones

📖 Saber

  • Pensamiento computacional: descomposición, abstracción, reconocimiento de patrones, algoritmia
  • Estructuras de control: secuencia, alternativa, iteración
  • Variables, listas y mensajes en Scratch
  • Eventos, sensores y bucles

🛠️ Hacer

  • Diseñar el algoritmo de un juego o animación antes de programarlo
  • Usar variables y listas para guardar el estado del juego
  • Depurar errores de un programa sencillo

✅ Evaluar

  • El programa funciona y resuelve la tarea pedida
  • Se usan estructuras de control adecuadas
  • El proyecto está documentado (qué hace, cómo se usa)
